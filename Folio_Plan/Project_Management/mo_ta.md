Mục tiêu website:
- Tạo trang folio chuẩn doanh nghiệp/ cá nhân - cấp lãnh đạo
- Màu sắc đen trắng chủ đạo, màu xám là màu bù, màu sắc khác làm điểm nhấn
- phong cách thiết kế minimal
- UX/UI hiện dại, tối ưu trải nghiệm người dùng
- Tối ưu SEO
- Tối ưu mobile
- Tối ưu tốc độ tải trang

Thư mục Folio_Plan là nơi chứa các file kế hoạch phát triển website đúng tiêu cuẩn đang triển khai cho nhánh marketing đang được phát triển

Yêu cầu phân nhánh rõ ràng thư mục nội dung triển khai và code cho web tránh nhầm lẫn và bị xâm phạm lẫn nhau.

File original_stitch.html được coi là style, màu sắc, bố cục tiêu chuẩn để triển khai cho tất cả các trang con của website.
